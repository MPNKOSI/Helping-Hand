What is HTML?

TML (HyperText Markup Language) is the standard language used to create and structure web pages.

It tells the browser:

  • What content to display
  • How content is structured
  • What is a heading, paragraph, image, link, button.
  • HTML is the foundation of every website.

How HTML Works

When you visit a website:

  • The browser downloads an HTML file.
  • It reads the tags.
  • It builds the page structure.
  • CSS styles it.
  • JavaScript makes it interactive.

Think of it like this:

  • HTML = Structure (Skeleton)
  • CSS = Design (Clothes)
  • JavaScript = Behavior (Brain)
