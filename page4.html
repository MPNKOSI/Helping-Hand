What You Need to Use HTML

A Device

  • Laptop (Recommended)
  • Desktop
  • Even a phone (but not ideal for real development)
  • Since you're serious about development, a Windows/Mac/Linux laptop is best.
Helpful Websites & Apps

Available on Andriod/IOS

  • Code Studio
  • icodethis.com
  • w3schools.com

A Code Editor (Very Important)

This is where you write your HTML code.

Recommended:

  • Visual Studio Code (VS Code), it's Free, Lightweight, Professional & Supports extensions

    • Other options:

  • Notepad (basic)
  • Sublime Text
  • WebStorm (paid)
  • But VS Code is the best choice for you.

A Web Browser

To open and test your HTML files.

Recommended:

  • Microsoft Edge

    • Other options:

  • Google Chrome
  • Mozilla Firefox
  • These browsers read your HTML and display the webpage.

What You DON’T Need

Database ,Backend, hosting, Frameworks, Internet (for basic HTML)

HTML is simple.

  • Git → Version control
  • Hosting → Put site online
